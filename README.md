# 🚦 Smart Traffic Management System using Computer Vision

An AI-based traffic monitoring and analysis system that detects and counts vehicles in real time using YOLO object detection and computer vision techniques. The system analyzes traffic density and provides adaptive signal timing recommendations to support intelligent transportation and smart city applications.

---

## 📌 Project Overview

Traffic congestion is a major issue in urban environments. This project proposes an intelligent traffic monitoring solution that uses computer vision to automatically detect vehicles from video streams and estimate traffic density. Based on congestion levels, the system recommends optimized signal timing to improve traffic flow efficiency.

---

## ✨ Key Features

- Real-time vehicle detection using YOLO (You Only Look Once)
- Traffic density classification (Low / Medium / High)
- Adaptive traffic signal timing recommendation
- Data logging for traffic analysis
- Visual display with bounding boxes and statistics
- Scalable architecture for smart city integration

---

## 🛠 Technologies Used

- Python
- OpenCV
- YOLOv8 (Ultralytics)
- NumPy
- Pandas

---

## ⚙️ System Architecture

1. Video Input Acquisition  
2. Frame Processing using Computer Vision  
3. Vehicle Detection using YOLO Model  
4. Traffic Density Estimation  
5. Signal Timing Decision Logic  
6. Output Visualization and Data Logging
